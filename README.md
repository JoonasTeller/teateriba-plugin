# Minu loodud teateriba plugin

### Mida see teeb?
Lõin plugina, mis kuvab veebilehe ülaservas silmapaistva teateriba olulise info edastamiseks. Lisasin ka JavaScripti põhise sulgemisnupu, et kasutaja saaks teate soovi korral eemaldada.

---

### Kuidas ma selle paigaldasin?

1. **Lõin faili** - Tegin kataloogi `wp-content/plugins/` uue PHP-faili koos vajaliku päisega.
2. **Lisasin koodi** - Kirjutasin funktsiooni, mis sisaldab teate HTML-i ja CSS-kujundust.
3. **Sidusin süsteemiga** - Kasutasin `wp_footer` konksu, et kood ilmuks automaatselt kõikidel lehtedel.
4. **Aktiveerisin** - Lülitasin plugina WordPressi haldusliideses sisse.
